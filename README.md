# differential-gene-expression-using-DESeq2

This project performs differential gene expression analysis using RNA-seq data with DESeq2 in R.
The goal is to identify genes that are significantly upregulated or downregulated under differential experimental conditions.
The analysis includes normalization, statistical testing, and visualization of results.

## Tools and libraries

## Workflow
- Data import
- Normalization
- Differential expression analysis using DESeq2
- Statistical filtering of significant genes
- Visualization (PCA plot, volcano plot, heatmap)
